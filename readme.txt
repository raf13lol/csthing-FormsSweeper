If you wish to change the width, height or bombs in the grid, here's how:

Run the game from the command line, but at the end put "-w=GRID_WIDTH_HERE" and/or "-h=GRID_HEIGHT_HERE" and/or "-b=BOMBS_TO_MAKE".
For example:

`FormsSweeper.exe -w=20 -h=20 -b=50`

Have fun.
